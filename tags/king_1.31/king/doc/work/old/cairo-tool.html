The C-alpha Hinges tool

The Hinges tool is used to adjust segments of protein backbone without disturbing the surrounding structure. It allows you to select a region of backbone that joins two alpha carbons, and then rotate that segment of backbone around an axis drawn between those C-alphas. Using Hinges requires that chiropraxis.jar be present in the same folder as king.jar.

Create a hinge by control-clicking the alpha carbons that act as its endpoints or anchors. To use the dial control, click and drag the pointer to the desired value. For finer control you can shift-drag along an imaginary slider (this behavior is somewhat like the virtual dials in O). To reset the dial to its starting value, double click the dial face. The dial can also be adjusted from the graphics window using the mouse wheel or arrow keys; see below.

The Hinges panel provides feedback on the geometric quality of the current model, displaying residues names, deviation of the tau angle (N-CA-C) from ideality, and the position of the residue on the Ramachandran plot (phi, psi, and favored/allowed/outlier). Deviant geometry is highlighted in red.

This tool relies on the naming convention established by Prekin to find the appropriate part of the PDB structure from the point ID encoded in the kinemage. Future versions may use a spatial relationship instead, but that is not yet implemented.

Hinges is able to call Probe to dynamically generate contact dot surfaces as the model is moved. Using this feature requires that your OS can find Probe (typically, that Probe be on your PATH and be named 'probe').

Hinges has not yet been published. When it is, a citation will appear here.

More documentation is planned for future releases.

Mouse clicks

NormalPick (mark point and show label; make measurement)
With ShiftCenter on selected point (pickcenter)
With Ctrl(On a C-alpha) Select anchor for hinge
(Elsewhere) Clear current selection
With Shift+Ctrl-

Mouse drags

NormalRotate around X and Y axes
Z-rotate (pinwheel) near top of screen
With Shift(Up/down) Adjust zoom
(Left/right) Adjust clipping
With CtrlTranslate in X-Y plane (flatland)
Z-translate near top of screen
With Shift+CtrlRotate around Y axis only

Mouse wheel / Up & Down arrow keys

NormalAdjust dial (angle of rotation)
With ShiftAdjust clipping
With CtrlAdjust zoom
With Shift+Ctrl-

For multibutton mice

Normal means the left-most mouse button.
The middle mouse button acts like Normal+Ctrl.
The right mouse button acts like Normal+Shift.
The left and right mouse buttons together act like Normal+Shift+Ctrl (only when dragging).
