The Edit Properties tool

This tool is just like Navigate, except that picking a point opens the editing window for that point. To pick a point without editing it, use Ctrl+<click>. You will need to activate the Explore | Superpick option to edit unpickable points. You can also relocate points by Ctrl-clicking and dragging them to a new location.

Mouse clicks

NormalPick point and show properties for editing
With ShiftCenter on selected point (pickcenter)
With CtrlPick (mark point and show label; make measurement)
With Shift+Ctrl-

Mouse drags

NormalRotate around X and Y axes
Z-rotate (pinwheel) near top of screen
With Shift(Up/down) Adjust zoom
(Left/right) Adjust clipping
With CtrlTranslate picked point in X-Y plane
Normal "flatland" if no point picked
With Shift+CtrlRotate around Y axis only

Mouse wheel / Up & Down arrow keys

NormalAdjust zoom
With ShiftAdjust clipping
With Ctrl-
With Shift+Ctrl-

For multibutton mice

Normal means the left-most mouse button.
The middle mouse button acts like Normal+Ctrl.
The right mouse button acts like Normal+Shift.
The left and right mouse buttons together act like Normal+Shift+Ctrl (only when dragging).
