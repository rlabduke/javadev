The Emphasize tool

The Emphasize tool is used to highlight one list out of many (presumably similar) lists in a single subgroup. In fact, two different highlights can be maintained for easy comparison of lists. Each of the two highlights can be enabled or disabled with the checkboxes; by default only the primary one is enabled.

This tool was developed for use with Chiropraxis. If you find other uses for it, let us know...

Mouse clicks

NormalSelect subgroup from which to emphasize lists
With ShiftCenter on selected point (pickcenter)
With CtrlPick (mark point and show label; make measurement)
With Shift+Ctrl-

Mouse drags

NormalRotate around X and Y axes
Z-rotate (pinwheel) near top of screen
With Shift(Up/down) Adjust zoom
(Left/right) Adjust clipping
With CtrlTranslate in X-Y plane (flatland)
Z-translate near top of screen
With Shift+CtrlRotate around Y axis only

Mouse wheel / Up & Down arrow keys

Move slider for list #2 (right slider)
NormalMove slider for list #1 (left slider)
With Shift
With CtrlAdjust zoom
With Shift+CtrlAdjust clipping

For multibutton mice

Normal means the left-most mouse button.
The middle mouse button acts like Normal+Ctrl.
The right mouse button acts like Normal+Shift.
The left and right mouse buttons together act like Normal+Shift+Ctrl (only when dragging).
